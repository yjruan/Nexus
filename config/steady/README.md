The files `WiRX.gY` correspond to MD configurations at varing shear rates $Wi_\text{R} =$ `X` and strians $\gamma =$ `Y`, respectively, 
for the standard Kremer–Grest model with chain length $N = 500$.
To reduce the file size, only the coordinate information is retained in these configuration files.
These configurations correspond to systems that have already steady state under shear.
These reduced configurations are compatible with Kröger's [Z1+ code](https://github.com/mkmat/Z1plus-code/) for primitive path analysis, and the corresponding results are summarized in the table below.
As shown in the table, at steady state, the number of entanglements remains nearly unchanged over the range of $Wi_\text{R}$ examined.

| 4.8 | 12.960 |
