The files `data.0` to `data.5` correspond to MD configurations at strains 
$\gamma = 0, 1, 2, 3, 4,$ and $5$, respectively, for the standard Kremer–Grest model 
with chain length $N = 500$ under shear at $Wi_\text{R} = 59.2$.
To reduce the file size, only the coordinate information is retained in these configuration files. 
These reduced configurations are compatible with Kröger's [Z1+ code](https://github.com/mkmat/Z1plus-code/) for primitive path analysis, and the corresponding results are summarized in the table below.
As shown in the table, during startup shear, the number of entanglements gradually increases with increasing strain.

| Strain $\gamma$ | $Z_\text{k}$ |
|---:|---:|
| 0 | 13.617 |
| 1 | 13.792 |
| 2 | 14.291 |
| 3 | 14.875 |
| 4 | 15.366 |
| 5 | 15.522 |
